# gpu_tutorial
Several simple code to test my knowledge of CUDA. It is also helpful in testing
the function of individual GPUs.


### Compile and Run

Testing simple GPU functionality
```
ml load cuda11.8/toolkit/11.8.0
make
./hello_gpu
```
